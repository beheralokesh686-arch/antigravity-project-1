# 🎓 EduTrackX - Student Result Management System

## 📌 Project Overview
EduTrackX ek web-based application hai jo students ke marks manage aur result generate karne ke liye use hota hai.

## 🚀 Features
- Admin login system
- Student record management
- Marks entry (Math, English, Physics, Chemistry, etc.)
- Automatic result & percentage calculation
- Clean UI with Flask backend

## 🛠️ Technologies Used
- Python (Flask)
- MySQL Database
- HTML, CSS, JavaScript
